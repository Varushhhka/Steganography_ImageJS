Данный проект реализует полный цикл исследования в области цифровой стеганографии:
1. Встраивание данных: Детерминированный алгоритм пространственного LSB-встраивания с побитовой модификацией пикселей(LSB) в форматах без потерь (PNG) с помощью ImageJS.
2. Оценка незаметности: Расчёт объективных метрик качества (PSNR, MSE, SSIM) и визуальный анализ артефактов внедрения.
3. Стегоанализ:
   а) Автоматизированный анализ с использованием фреймворка Aletheia (методы WS и EffNetB0).
   б) Собственная реализация классических статистических методов (хи-квадрат тест и RS-анализ).
4. Сравнительный анализ: Оценка эффективности методов детектирования при различных объёмах внедряемой информации.
